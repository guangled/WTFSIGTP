# WTFSIGTP
Where the fuck should I go to play?

# Prerequisite
1. npm

# Installation
1. git clone https://github.com/yihan940211/WTFSIGTP.git
2. cd WTFSIGTP
3. npm Installation
4. npm start
5. visit localhost:9090. You can change the port number in the webpack.config.js file.
